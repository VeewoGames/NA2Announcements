# November 18th Update Announcement

Note: Updating the version may cause errors in saved game data. Please complete your current gameplay session before updating.

# Beta Branch Update

### Bug Fixes

* Fixed display issues with slot machine and bingo machine graphics
* Resolved error messages appearing during bar NPC dialogues
### New Features

* Added new art assets for the DLC button on the main menu
### Other Updates

* Removed powerUp event callback deletion for BattleNeon nodes in Challenge Rooms
* Improved network synchronization performance during Hermes boss fights
* Adjusted Mira's skill UI display logic - now only appears when skills are active
### How to Switch to the Beta Branch

In your Steam Library >>> Right-click on "Neon Abyss 2" >>> Properties >>> Betas >>> Beta (no password required)

Please note that players on different branches cannot play together due to version differences.
