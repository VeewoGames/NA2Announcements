# October 23rd Update Announcement

Note: Updating the version may cause errors in saved game data. Please complete your current gameplay session before updating.

# Beta Branch Update

### Bug Fixes

* Fixed an issue where Relics would disappear when thrown during multiplayer sessions
* Fixed an issue where some bosses weren't granting EXP upon defeat
* Fixed an issue where ghost enemies sometimes wouldn't Attack during multiplayer
### Experience Improvements

* Optimized Challenge Room gameplay experience with smoother difficulty progression
### How to Switch to the Beta Branch

In your Steam Library >>> Right-click "Neon Abyss 2" >>> Properties >>> Betas >>> Beta (no password required)

Please note that multiplayer between different branches is not possible due to version differences.
