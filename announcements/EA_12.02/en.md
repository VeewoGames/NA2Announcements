# Stable Branch Update

### Bug Fixes

* Fixed a display error with Slot Machine images
* Fixed incorrect behavior of the God of Guns
* Fixed an error caused by moving while upgrading weapons
* Fixed an error caused by clicking "Continue Game" multiple times on the results screen
* Fixed an issue where picking up items in the Challenge Room would cause other items/weapons to disappear
### How to Switch to the Beta Branch

Steam Library >>> Neon Abyss 2 >>> Right-click >>> Properties >>> Betas >>> Beta (No password required)

Please note that multiplayer is not available between different branches due to version mismatches.
