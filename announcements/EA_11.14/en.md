Note: Updating the version may cause errors in saved game data. Please complete your current run before updating.

# Beta Branch Update

### Update Highlights

November update content is now available for testing in the Beta branch

**BOSS**: Hermes joins the Abyss

**Agent**: New Agent Mira joins the Abyss

**New Weapon**: BLOODBOND (Charging at right moment consumes all hearts and shields to summon the reusable Blood Servant. Weapon is enhanced when at Death's Door.)

### Art

* Visual updates for original bosses
### Bug Fixes

* Fixed issue where players couldn't exit Soul Room after using dice to reroll items
### How to Switch to the Beta Branch

In your Steam Library >>> Right-click "Neon Abyss 2" >>> Properties >>> Betas >>> Beta (no password required)

Please note that multiplayer between different branches is unavailable due to version differences.
