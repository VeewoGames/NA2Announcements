# October 27th Update Announcement

Note: Updating the version may cause errors in saved game data. Please complete your current gameplay session before updating.

# Beta Branch Update

### Bug Fixes

* Fixed an issue with Perfect Ruby causing abnormalities when continuing gameplay
* Fixed Mantra causing abnormal display of minimap icons
### Other Updates

* Optimized the deployment logic for Bouncing Mines - now requires manual triggering during flight
* Added a new batch of Attack-type items
* Updated SEASPELL attack module
* Dance King modification: When used, consumes Crystals to boost weapon damage until combat ends. Also generates a dance platform - interact with it to dance.
* New Curio added: Drifting Bottle - consumes Crystals to store unopened Chests (max 5). Using it again releases all stored Chests.
### How to Switch to the Beta Branch

In your Steam Library >>> Neon Abyss 2 >>> Right-click >>> Properties >>> Betas >>> Beta (no password required)

Please note that multiplayer between different branches is unavailable due to version differences.
