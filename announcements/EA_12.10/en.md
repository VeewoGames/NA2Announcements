# December 10th Update Announcement

Note: Updating the version may cause errors in saved game data. Please complete your current gameplay session before updating.

# Default Branch Update

-----

### Bug Fixes

* Fixed multiplayer issues with Street Glory.
* Fixed an issue where teammates might experience a black screen in multiplayer mode.
* Fixed abnormal display in character selection interface where characters might appear duplicated.
* Fixed terrain generation issues when continuing a game (unchanged terrain).
* Fixed room generation logic to prevent two rooms from overlapping at the same position.
* Fixed interaction issues after picking up Hexjars.
* Fixed error reports in APOLLO boss room.
* Fixed some display errors in the Codex and ban interface.
* Adjusted damage and power calculation formulas for Hatchmon-type weapons to prevent negative values.
### Experience Improvements

* Optimized the ban interface to automatically filter and hide characters' initial weapons.
-----

### How to Switch to the Beta Branch

In your Steam Library >>> Neon Abyss 2 >>> Right-click >>> Properties >>> Betas >>> Beta (no password required)

Please note that players on different branches cannot play together due to version differences.
